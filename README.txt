# Number Guessing Game

This is a simple command-line number guessing game written in Python.

## How to Play

Run the `main.py` script. The program will randomly select a number between 1 and 100, and you have to guess it. After each guess, you will receive a hint whether your guess was too low, too high, or correct.

## Requirements

- Python 3.x

## Usage

```bash
python main.py
```

Enjoy the game!
